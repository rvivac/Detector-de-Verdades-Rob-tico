# Detector de Verdades Robótico (PWA)

Este repositório publica um app web responsivo (PWA) em **arquivo único** (`index.html`) para simular um “Detector de Verdades” com visual sci‑fi/cyberpunk e sons via Web Audio API.

## Como abrir localmente

1. Baixe o repositório.
2. Abra o arquivo `index.html` no navegador.

Observação: alguns recursos de PWA (instalação/offline) funcionam melhor quando hospedado em HTTPS.

## Hospedar no GitHub Pages

1. Vá em **Settings** do repositório.
2. Entre em **Pages**.
3. Em **Build and deployment**:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Salve. Em alguns segundos/minutos o GitHub vai mostrar a URL do seu site.

## Instalar como aplicativo no celular (PWA)

- **Android (Chrome/Edge)**: abrir o link do Pages → menu ⋮ → **Instalar app** / **Adicionar à tela inicial**.
- **iPhone (Safari)**: abrir o link do Pages → **Compartilhar** → **Adicionar à Tela de Início**.
